# Sales Dashboard Project

This project is a monthly sales analysis dashboard created in Microsoft Excel.

## Features
- Daily sales entry sheet
- Automatically calculated total amount
- PivotTable summary
- Top-selling products chart
- Clean and interactive dashboard

## Tools Used
- Microsoft Excel (PivotTables + Charts)

## Objective
To track and visualize product-wise sales performance.
